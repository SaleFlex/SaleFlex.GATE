SaleFlex.GATE

The central Django and Django REST Framework hub for the SaleFlex ecosystem: stores, terminals, restaurant flows, mobile apps, and optional third-party systems — aligned with SaleFlex.PyPOS and related clients.

Create an account Log in

What GATE does

Multi-tenant hub

Companies, stores, and terminal profiles as the basis for data isolation and API access.

REST APIs

Versioned JSON for devices and mobile; merchant token auth for POS-style integrations.

POS-aligned domain

Merchants, closures, warehouses, customers, and reference data under active development.

Integration gateway

Front ERP, loyalty, campaign, and payment adapters so edge apps stay thin.

Documentation

Architecture and tenancy are described in the docs/ folder and the project README. This site is a starting point for browser-based access; ERP-style screens will grow over time.
